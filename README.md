# vue-skeleton

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
nano vue.config.js
// vue.config.js
module.exports = {
  publicPath: '/LAB8_10/dist/'
}
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Open project in browser at URL
```
http://localhost/LAB8_10/dist/
```
